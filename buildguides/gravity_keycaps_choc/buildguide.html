# Gravity Keycaps Choc v1 ビルドガイド

  • 下記の図は裏側からみた状態です
  • 図の状態が再現できれば正しく取り付けられます
  • 各タイプごとに配置を行います（下記参照）
  • 一番右側が、表側から見たときの一番左になります
  • 右側から取って、左側に配置をしていくと良いかと思います
  • 上のキーはbat43用の真ん中に使用するキーです。マーク有り、無しのどちらでもOKです
  • 親指キーの真ん中は1.25Uと1Uの物がありますが、お好みで使ってください

# Highタイプ

# 参考手順

# マークが無いものを集めます

  • （黄緑）切り欠きのあるものは中央のキーになります。切り欠きを上にして配置してください
  • （紫）四方が囲まれているものは、親指キーになります

# マークがあるものを分類します

  • 丸いマークの大きさ（大・小）、数（1〜4個）、位置で配置がわかるようになっています
  • 各マークは外側に位置するようになっています
  • 上段を揃えます
    • 大きいマークの物が上段になります
      • （赤）マークが中央にあるもの
      • （オレンジ）マークが角にあるもの
  • 中段を揃えます
    • （水色）真横に3つマークがあるもの
  • 下段を揃えます
    • 残った小さいマークのものです
      • （茶色）4つマークは内側
      • （青）2つマークは外側に配置します
      • （黄）残りはマークが下になるように配置します
