# Gravity Keycaps Choc v2 ビルドガイド

# Highタイプ

図は裏側からみた状態です。
3行6列、親指キーが3つ程度の分割キーボードを想定しています。
使用予定のキーボードに合うように適宜置き換えてください。

キーキャップは刻印のあるものとないものがあります。
刻印がないものは2種類あります。

# Centerキー

刻印がなく、壁が一面だけ抜けているものがCenterキーです。
主にホームポジションに置かれることを想定していますが、汎用的にも使えます。
表側に突起のついているものはホーミングキーとして使用できます。
お好みで使ってみてください。


# 親指キー

刻印がなく、壁に囲まれているのが親指キーです。
1.25Uのキーはwings42やbat43を想定して、左右どちらかに偏っています。
お好みによって、一番端のキーとしてお使いください。


# 刻印ありのキー

その他のキーは下図のように刻印があります。
この刻印に従って、一番上の図のように配置してください。

刻印は3種類の記号から成り立っています。

ポジション + 高さ + 角度

CH15 = Corner + High + 15度 
 H15 =          High + 15度 
SL15 = Side   + Low  + 15度 
CL10 = Corner + Low  + 10度 
CL20 = Corner + Low  + 20度 
 L15 =          Low  + 15度

Corner用のキーは左右反転しているものがありますが、刻印では区別できないのでキーボードに合った向きでご使用してください。
