# Gravity Keycaps MX ビルドガイド

  • 一番右側が、表側から見たときの一番左になります
  • 右側から取って、左側に配置をしていくと良いかと思います

# 参考手順

# マークが無く、縁が囲われているものを集めます

こちらは全て親指キーになります。

親指キーは3種類あります。
・1.25Uキー
・ひとまわり小さい1Uキー
・1Uキー

ひとまわり小さいキーはお好みでお使いください。
Claw44の内側に普通の1Uを配置すると、若干邪魔になるので小さめで作ってあります。

一個、BOX状のキーがありますが、こちらはおまけです。
お好みで使ってください。

# マークが無く、切り欠きのあるものを集めます

こちらは中央のキーになります。
切り欠きを上にして配置してください。
表側、突起がついているものが2つあるので、お好みでホーミングキーとしてお使いください。

# マークがあるものを分類します

  • 丸いマークの大きさ（大・小）、数（1〜4個）、位置で配置がわかるようになっています
  • 各マークは外側に位置するようになっています
  • 上段を揃えます
    • 大きいマークの物が上段になります
      • （赤）マークが中央にあるもの
      • （オレンジ）マークが角にあるもの
  • 中段を揃えます
    • （紫）真横に3つマークがあるもの
  • 下段を揃えます
    • 残った小さいマークのものです
      • （白）4つマークは内側
      • （青）2つマークは外側に配置します
      • （黄）残りはマークが下になるように配置します
