# OLED

# OLEDモジュールのはんだ付け

  • OLEDモジュールの袋に一緒に入っているピンヘッダは大きめなので、別の袋に入っている小さめのピンヘッダを使用してください

  • OLEDモジュールとピンヘッダを組み立てます

  • ピンヘッダをはんだ付けします

  • 液晶についている保護シールは剥がしてください

# ピンソケットのはんだ付け

  • こちらのスペースに取り付けます

  • 「OLED」とシルクで書かれている所にピンソケットを挿します

  • マスキングテープで固定します

  • 裏側からはんだ付けをします

# OLEDの取り付け

  • ピンソケットにOLEDを差し込みます

  • [[キーボード/sandbox/ビルドガイド詳細/動作確認とキーマップ|動作確認とキーマップ]]を参考に、OLED用のファームウェアを書き込んでください

  • USBケーブルを挿すと、OLEDが点灯します
