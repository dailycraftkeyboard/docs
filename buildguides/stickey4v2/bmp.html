# BMP

# プレートの取り外し

  • プレートを外して、基板を取り出します
    ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-1.jpeg]]

# コンデンサの取り付け

  • パッドの片側に予備ハンダをつけます ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-2.jpeg]]

  • 予備ハンダを利用してコンデンサをはんだ付けして固定します。もう片方もはんだ付けします。 ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-3.jpeg]]

# スライドスイッチの取り付け

  • スライドスイッチを載せます ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-4.jpeg]]
  • 裏返して、はんだ付けをします ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-5.jpeg]]
  • プレートに干渉するので足を短く切ります ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-6.jpeg]]

# 電池ケースの取り付け

  • 電池ケースを載せます ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-7.jpeg]]
  • テープなどで固定します ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-8.jpeg]]
  • 裏返して、はんだ付けをします ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-9.jpeg]]

# BMP、電池の取り付け

  • BMPはコンスルー対応のため、はんだ付けは不要です（はんだ付けしても良いです）
    13ピンコンスルーを使用してください
    ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-10.jpeg]]

  • BMPを取り付けます
    ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-11.jpeg]]

  • 電池(CR2032)を取り付けます
    ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-12.jpeg]]

  • スイッチは上側がONになります
    ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-13.jpeg]]

  • エンコーダーをつける場合は、電池の上のエンコーダの足を短くするか、カプトンテープなどで保護してください。足を曲げていればショートしないはずですが、念の為の処置をお願いします。 ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-14.jpeg]]

# プレート組み立て

  • プレートを組み立て直せば完成です！ ![[キーボード/stickey4/ビルドガイド(v2)/assets/i9-15.jpeg]]

# ファームウェア

  • ファームウェアの設定
    • BMPのドキュメントを参照ください
      • 自作キーボードキットを無線化する (opens new window)
    • WebConfiguratorにstickey4が登録されているので、その設定を書き込んでください
    • キーマップはPro Micro同様にRemapから設定してください
  • ロータリーエンコーダのキーマップはBMPの設定ファイルで行ってください
    • 参考エンコーダ設定ファイル
      • https://github.com/yfuku/stickey4/blob/main/bmp/
    • ※BMPだとRemapではロータリーエンコーダの設定はできません
