# 変更履歴

# v2(rev2)

  • 2022/02/XX Daily Craft Keyboardで販売開始
  • Pro Micro Type-Cに対応
  • それに伴い外形変更
  • 拡張プレートの形状も変更
  • v1の拡張プレートは使用できますが、上部に少し隙間ができます
  • ファームウェアはrev2のまま変更はありません

# v1(rev2)

  • 2021/08/01 Daily Craft Keyboardで発売開始
  • キー拡張プレートの仕様変更
    • 6ピンコンスルーによる接続から、スペーサーによる接続に変更
    • beta版用のプレートは使えません
  • ロータリーエンコーダ拡張プレートの仕様変更
    • エンコーダのクリックに対応
    • beta版用のプレートは使えません
  • BMPバッテリープレートの仕様変更
    • スペーサーによる接続から、2ピンコンスルーで電源供給するように変更
    • beta版用のプレートは使えません
  • ファームウェアをrev2にアップデート
    • rev1ではキー拡張は別ファーム、エンコーダはソースからのコンパイルが必要でしたが、同一ファームウェアでRemap/VIAから設定できるように改善しました

# beta 2(rev1)

  • 2021/05/31 BOOTHで発売
  • BMPバッテリープレートに対応(rev2用は使用できません)
  • (拡張プレートはbeta対応のみ使用できます)

# beta 1(rev1)

  • 2021/05/01 BOOTHで発売
  • (BMPバッテリープレートは使用できません)
  • (拡張プレートはbeta版のみ使用できます)
