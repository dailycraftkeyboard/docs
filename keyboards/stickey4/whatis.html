# stickey4とは？

シンプルな4キーマクロパッドです。

# 特徴

# 組み立て

  • ダイオードを使用していないので、はんだ付けが簡単
  • 部品点数が少ないので組み立てしやすい
  • スイッチソケットは非対応（キースイッチに直接はんだ付けをします）

# キープロファイル

  • Choc v1スイッチ専用
  • 狭ピッチ（17x18mm）
    • 一般的なキースイッチの間隔19x19mmより狭くなっています

# 拡張プレート

  • 上部のプレートを変更することで、ロータリーエンコーダを使うこともできます

# 無線対応

BLE Micro pro (opens new window)を使用することで、無線接続も可能です。

# ファームウェア

  • QMK、Remap、VIA対応
