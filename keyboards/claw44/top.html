# Claw44

自作キーボードキット Claw44の紹介ページになります。
まずは、Claw44とは？をお読みください。

# 購入先

  • オフィシャルショップ
    • Claw44 v3 | Daily Craft Keyboard (opens new window)
  • 遊舎工房（委託販売）
    • 【委託】Claw44 v3 | 遊舎工房ショップ (opens new window)

# 組み立て

Claw44はご自身で半田付けなどをして組み上げる自作キットになります。
ご購入の際にはビルドガイドをよく読み、必要な道具や作業を把握してください。
組み立てに自信のない方は、組み立てサービスを開始したので、こちら (opens new window)からキット本体と一緒にご購入ください。

# ビルドガイド

  • Claw44 v3 ビルドガイド

# 作りやすさ

  • 左右独立基板になったため、部品を逆につけるなどミスが発生しにくい
  • ProMicroはコンスルー付きなので取り外し可能
  • キースイッチはソケット対応なので取り外し可能
  • 部品同士が重ならないシンプルなつくり
  • ダイオードはスルーホール用が同梱されてますが、表面実装タイプも使用可能
