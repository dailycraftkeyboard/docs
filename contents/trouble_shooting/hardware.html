# ハードウェア

# BMPの特定のピンが機能しない

  • stickey4で使っていたBMPをClaw44 v2に持ってきた
  • web configratorで一通り書き込んだ
  • 特定のrowだけ機能しない
  • stickey4のENCODER.JSNが残っていて、そこに設定されたピンが無効になっていた
  • ENCODER.jsnを無機能なものに変更したたら治った

# Pro Microのスルーホール不具合

  • 組み立てたがcolが3つ分反応しない。
  • Pro Microを交換しても発生するので、基板側の問題か？
  • テスターでチェックすると、パッドには導通するが、コンスルーには導通していなかった
  • プロマイクロの穴がなんらかの理由により塞がれていている？
  • 穴を精密マイナスドライバーの小さいやつででぐりぐりしたら通るようになった
  • コンスルーが反応しなくなるので削りすぎに注意
  • 爪楊枝とかでやった方が良かったかも
  • 基板の製造工程というより、はんだのフラックスが飛んで付着してしまったのかもしれない
